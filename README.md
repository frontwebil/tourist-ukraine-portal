# 🇺🇦 Найвідоміші туристичні місця України

**Курсова робота** — Інформаційний портал на основі web-технологій (3-й ступінь складності)

## 📋 Опис

Інформаційний портал, присвячений найвідомішим туристичним місцям України: замки, гори, природні парки, історичні міста та культурні пам'ятки.

| **Next.js 15** | React-фреймворк (SSR, App Router) |

| **Supabase** | PostgreSQL хмарна БД |

| **TypeScript** | Типізація |
| **Swiper** | Слайдер |
| **CSS Grid** | Сіткова розмітка |
| **Flexbox** | Гнучка розмітка |
| **Медіазапити** | Адаптивний дизайн |

## 🚀 Запуск

### 1. Встановлення залежностей

```bash
npm install
```

### 2. Налаштування Supabase

1. Створіть проєкт на [supabase.com](https://supabase.com)
2. Скопіюйте `.env.example` → `.env`
3. Вставте свої значення `DATABASE_URL` та `DIRECT_URL` з Supabase Dashboard → Settings → Database

```bash
cp .env.example .env
# Відредагуйте .env — вставте свої рядки підключення
```

### 3. Створення таблиць та наповнення БД

```bash
npx prisma db push
npm run db:seed
```

### 4. Запуск dev-сервера

```bash
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000)

## 📱 Адаптивний дизайн

Портал повністю адаптивний:
- **Десктоп** (>992px) — повна навігація, 3-колонкова сітка
- **Планшет** (577–992px) — бургер-меню, 2-колонкова сітка
- **Мобільний** (<576px) — одноколонкова сітка, компактний слайдер

## 📄 Сторінки

1. **Головна** — слайдер з 5 туристичних місць + 10 останніх новин
2. **Про сайт** — інформація про портал, статистика, використані технології
3. **Галерея** — фото з фільтрацією по категоріях та лайтбоксом
4. **Новини** — всі новини з детальним переглядом
5. **Контакти** — форма зворотного зв'язку + Google Maps
